# 🌸 心情花园

一款治愈风格的心情日志应用，帮助你记录每日情绪，培育属于自己的心灵花园。

## ✨ 功能特点

- 🎨 **多种心情选择** - 8种不同情绪（开心、难过、生气、平静、兴奋、疲惫、焦虑、充满希望）
- 🌷 **独特花朵映射** - 每种心情对应一朵风格各异的花
- 💬 **AI对话模式** - 可以直接与AI对话，分享你的感受
- 📝 **直接写日记** - 也可以选择直接写下自己的日记
- 🤖 **智能总结** - AI会根据对话内容生成情绪日记
- 🏡 **心灵花园** - 所有记录的花朵会在花园中绽放
- 📱 **本地存储** - 所有数据保存在本地，隐私安全
- 📅 **回顾功能** - 点击任意花朵可查看当时的心情和日记

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 🎮 使用说明

### 记录心情

1. 在首页选择今天的心情（点击对应的心情选项）
2. 选择记录方式：
   - **写日记模式**：直接在文本框中写下你的想法
   - **AI对话模式**：与AI聊天，分享你的感受
3. 点击"在花园里种下一朵花"按钮保存

### 查看花园

- 所有记录的心情都会在花园中以花朵的形式展示
- 每朵花代表一天的心情记录

### 回顾日记

- 点击花园中的任意花朵，可以查看：
  - 当天的日期
  - 心情状态
  - AI总结的日记
  - 完整的对话记录（如果是AI对话模式）
- 可以选择删除不再需要的记录

## 🛠️ 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TDesign Vue Next** - 企业级 UI 组件库
- **Vite** - 下一代前端构建工具

## 💡 设计理念

这个应用采用了治愈系的设计风格：

- **渐变色背景** - 柔和的紫色渐变，营造宁静氛围
- **圆润的卡片设计** - 温暖友好的视觉体验
- **流畅的动画效果** - 花朵绽放的动画，增添趣味性
- **直观的交互** - 简单易用的操作流程

## 📱 浏览器兼容

支持所有现代浏览器：
- Chrome (推荐)
- Firefox
- Safari
- Edge

## 🔒 隐私说明

- 所有数据仅保存在浏览器的本地存储中
- 不会上传到任何服务器
- 清除浏览器数据会丢失所有记录
