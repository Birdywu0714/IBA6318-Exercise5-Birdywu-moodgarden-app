# 🌸 心情花园

一款治愈风格的心情日志应用，帮助你记录每日情绪，培育属于自己的心灵花园。

## ✨ 功能特点

- 🎨 **多种心情选择** - 8种不同情绪（开心、难过、生气、平静、兴奋、疲惫、焦虑、充满希望）
- 🌷 **独特花朵映射** - 每种心情对应一朵风格各异的花
- 💬 **AI对话模式** - 可以直接与AI对话，分享你的感受
- 🤖 **真实AI回复** - 支持OpenAI、腾讯混元、智谱AI等多种AI服务
- 📝 **直接写日记** - 也可以选择直接写下自己的日记
- 🤖 **智能总结** - AI会根据对话内容生成情绪日记
- 🏡 **心灵花园** - 所有记录的花朵会在花园中绽放
- 📱 **本地存储** - 所有数据保存在本地，隐私安全
- 📅 **回顾功能** - 点击任意花朵可查看当时的心情和日记

## 🎮 使用说明

### 配置AI服务（可选）

为了获得更好的AI对话体验，你可以配置真实的AI服务：

1. 点击页面右上角的"⚙️ 配置AI"按钮
2. 选择AI服务提供商：
   - **OpenAI** - 功能强大，需要API密钥
   - **腾讯混元** - 国内服务，响应速度快
   - **智谱AI** - 优秀的国产AI服务
3. 输入对应的API密钥
4. 点击"保存配置"

**获取API密钥的方法**：
- OpenAI: 访问 https://platform.openai.com 注册并创建API密钥
- 腾讯混元: 访问 https://cloud.tencent.com/product/hunyuan 开通服务
- 智谱AI: 访问 https://open.bigmodel.cn 注册并获取密钥

> 注意：如果没有配置API密钥，应用会使用预设的通用回复，功能不受影响但互动性会降低。

### 记录心情

1. 在首页选择今天的心情（点击对应的心情选项）
2. 选择记录方式：
   - **写日记模式**：直接在文本框中写下你的想法
   - **AI对话模式**：与AI聊天，分享你的感受
3. 点击"在花园里种下一朵花"按钮保存

### 查看花园

- 所有记录的心情都会在花园中以花朵的形式展示
- 每朵花代表一天的心情记录

### 回顾日记

- 点击花园中的任意花朵，可以查看：
  - 当天的日期
  - 心情状态
  - AI总结的日记
  - 完整的对话记录（如果是AI对话模式）
- 可以选择删除不再需要的记录

## 🤖 AI服务说明

### 支持的AI服务

#### OpenAI GPT
- 模型：GPT-3.5-turbo / GPT-4
- 特点：功能强大，理解能力强，回复质量高
- 适用场景：需要深度对话和情感理解

#### 腾讯混元
- 模型：Hunyuan-Lite / Hunyuan-Pro
- 特点：国内服务，响应速度快，中文理解优秀
- 适用场景：日常使用，快速对话

#### 智谱AI GLM
- 模型：GLM-4
- 特点：优秀的国产大模型，中文表现突出
- 适用场景：中文对话，情感分析

### AI功能特点

- **情感化回复**：AI会根据你选择的心情状态调整回复风格
- **智能引导**：AI会通过提问引导你更深入地表达情感
- **日记总结**：AI会自动将对话总结成一篇完整的日记
- **温暖治愈**：所有AI回复都采用温暖、鼓励的语气

- **圆润的卡片设计** - 温暖友好的视觉体验
- **流畅的动画效果** - 花朵绽放的动画，增添趣味性

## 🔒 隐私说明

- 所有数据仅保存在浏览器的本地存储中
- API密钥存储在本地，不会上传到任何服务器
- 对话内容直接发送到AI服务提供商，符合其隐私政策
- 清除浏览器数据会丢失所有记录
